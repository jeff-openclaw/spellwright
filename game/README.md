# Spellwright — Unity Project

Create the Unity project IN this directory:

1. Open Unity Hub
2. New Project → 3D (URP) → Unity 2022.3 LTS
3. Set location to the PARENT of this folder, project name: `game`
   - Or create elsewhere and move contents here

**Important Unity settings:**
- Edit → Project Settings → Version Control → Mode: **Visible Meta Files**
- Edit → Project Settings → Editor → Asset Serialization: **Force Text**

These ensure clean git diffs and proper version control.
